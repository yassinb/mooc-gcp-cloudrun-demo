À propos du projet

Ce site sert de fil rouge pour comprendre les étapes d'un déploiement cloud : création d'une image Docker, stockage dans Artifact Registry, automatisation avec Cloud Build et déploiement sur Cloud Run.

Services GCP utilisés

Cloud Build

Automatise la construction de l'image Docker à chaque changement de code.

Artifact Registry

Stocke les images de conteneurs de manière versionnée et centralisée.

Cloud Run

Exécute l'application conteneurisée sans gestion de serveur.
